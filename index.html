Administratorem danych osobowych jest właściciel serwisu znalezieni.pl. Kontakt: kontakt@znalezieni.pl.

2. Jakie dane przetwarzamy

  • Dane analityczne: anonimowe dane o sesji (Google Analytics) — wyłącznie po wyrażeniu zgody. IP są anonimizowane.
  • Pliki użytkownika: CV i ogłoszenia są przesyłane do API wyłącznie w celu jednorazowej analizy. Serwis nie przechowuje tych plików po zakończeniu sesji. Dane są przesyłane do Anthropic PBC (USA) na podstawie umowy DPA z SCC.
  • Adres email (opcjonalnie): przetwarzany wyłącznie jeśli użytkownik poda go w formularzu dostępu próbnego lub skontaktuje się z nami.

3. Pliki cookie

  • Niezbędne: zapamiętanie limitu analiz (localStorage). Nie wymagają zgody.
  • Analityczne (za zgodą): Google Analytics (G-695CTY5XED) — anonimizowane, 26 miesięcy.

4. Odbiorcy danych

  • Google LLC (Analytics) — USA, SCC.
